Project Title: Adidas Shopping Cart DOM Workshop Description: A dynamic shopping cart interface built for Adidas products. This project demonstrates DOM manipulation using JavaScript to handle real-time price updates, item quantity adjustments, item deletion, and a "like" feature. Features:

Real-time Math: Total price updates instantly when quantities change.

DOM Manipulation: Add, remove, and modify HTML elements dynamically.

Interactive UI: Users can like items and delete them from the cart.

Theme: Custom "Pink & Black" neon aesthetic. Tech Stack: HTML5, Bootstrap 5, CSS3, JavaScript
